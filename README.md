### 使用方法
1. git clone https://github.com/Binaryify/NeteaseCloudMusicApi.git
2. node app.js
3. 以上是为网易云的后台打开方式另外还需要打开这边登录注册的后台api接口才能正常工作
4. git clone https://github.com/h-sina/vue3-ts-elementplus.git
5. npm run serve
