### 使用方法
#### 网易云音乐部署到本地方法
```
git clone https://github.com/Binaryify/NeteaseCloudMusicApi.git
node app.js
```
#### 后端部署 后端（nodejs）
```
https://github.com/h-sina/nodejs 下载 
node index.js 启动 
```
#### 启动部署 前端（vue+ts+elementplus）
```
git clone https://github.com/h-sina/vue3-ts-elementplus.git 
npm run serve 
```
